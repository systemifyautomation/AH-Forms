# AH-Forms

A premium SaaS-style client onboarding form application for Amington Hall, built with React and Tailwind CSS.

## Features

- 🚀 **Modern React App** - Built with Vite for fast development and optimized production builds
- 🎨 **Premium SaaS Design** - Beautiful, modern UI with gradient effects, glass morphism, and smooth animations
- 📝 **Multi-Step Form** - 4-step onboarding form with progress tracking
- 📱 **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile devices
- ⚡ **Fast & Lightweight** - Optimized for performance

## Tech Stack

- **React 19** - Modern React with hooks
- **Vite** - Next-generation frontend tooling
- **Tailwind CSS v4** - Utility-first CSS framework
- **ESLint** - Code quality and consistency

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/
│   ├── Header.jsx        # Navigation header
│   ├── Hero.jsx          # Hero section with stats
│   ├── Features.jsx      # Features showcase
│   ├── OnboardingForm.jsx # Multi-step form
│   └── Footer.jsx        # Footer with links
├── App.jsx               # Main app component
├── main.jsx              # App entry point
└── index.css             # Tailwind CSS styles
```

## Form Steps

1. **Personal Information** - Name, email, phone
2. **Company Information** - Company name, size, industry
3. **Project Details** - Project type, budget, timeline
4. **Additional Information** - How they heard about us, preferences
